Key Concepts

  • HTTP: Hypertext Transfer Protocol is the foundation of data communication on the web. It shows how messages are formatted and transmitted, and how web servers and browsers should respond to requests.
  • HTTPS: Secure Hypertext Transfer Protocol encrypts data using SSL/TLS to make sure privacy and security are in place. It protects sensitive information such as passwords, payment details, and personal data from tampering or eavesdropping.
  • Stateless: In a stateless protocol like HTTP, each request from a client to a server is treated as independent.
  • Cookies: Small pieces of data stored by the browser to maintain session state.
  • API: Known for enabling developers to access data and services from other apps, Application Programming Interface is a set of rules that allows different software applications to communicate with each other.
